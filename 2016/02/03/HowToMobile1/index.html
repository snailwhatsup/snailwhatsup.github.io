玩转手机软件开发1

code

软件开发，类似与建筑开发，过程分为两个步骤。
首先是设计，一个具体的软件，类似于一个建筑，需要设计师进行合理的设计；第二是，具体的开发，选择一门语言，将设计好的软件结构/建筑实现出来。
总结起来，就是：设计-建造
设计的过程千变万化，不同的软件往往有不同的功能，从而导致具体设计的不同。但是，建筑设计往往有一定的原则，如需要满足力学规律，同样，软件设计也是如此，需要遵循一定的原则，算法和数据结构的合理安排。另外，建筑设计往往有很多模式可以选择，软件也是如此，因此有人总结出23种设计模式，可以参考。
建造是一个技术活，木匠-瓦匠不同专业技术工种相互配合才能建好一套房子，同样，软件的建造也需要不同工具互相配合，基本的工具是选择合适的编程语言，一个成熟编程语言往往包含很多有效的特性和library。除了编程语言，还有一些编程IDE环境，类似于工具箱。技术活一个特点就是要练习，熟能生巧。

手机软件的开发：
1，设计的原则是什么(数据结构和算法有什么特点)？设计的模式有哪些(是否有官方的指南)？
2，建造的语言是什么？建造的IDE环境是什么？
3，我想开发的软件是什么？软件有什么功能？这些功能将如何设计实现？
4，这个软件选用什么语言开发？在什么IDE工具中开发？使用语言的具体library有哪些？

手机软件的特点是它有一个界面，是使用手指来交互的,GUI的数据结构/操作；另外，手机需要联网，因此它还需要有网络的结构/操作。
设计的模式有官方的UI 指南，有一些通用模式可以套用。
语言，Android-java,iphone-OC/swift,Web语言及其混合编程。
环境很多，Eclipse,netbeans,xcode,intellij,phonegap。
